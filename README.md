# 🌉 MCBridge - 极简我的世界中转联机模组

[![Minecraft Version](https://img.shields.io/badge/Minecraft-1.20.1-blue.svg?style=flat-square)](https://www.minecraft.net/)
[![Forge Version](https://img.shields.io/badge/Forge-47.2.0-orange.svg?style=flat-square)](https://files.minecraftforge.net/)

**MCBridge** 是一款专为《我的世界》设计的自动化内网穿透联机工具。它集成了高性能的 frp 技术，让您可以一键开启公网联机，彻底告别繁琐的内网映射步骤。

---

## 🚀 核心功能

- **⚡ 一键局域网映射**：在“对局域网开放”界面一键开启，自动获取外网地址。
- **🤖 自动化多人游戏连接**：进入“多人游戏”列表时自动在后台建立加速链路。
- **🛡️ 安全加密机制**：采用环境变量传递 Token，配置文件即用即删，严防信息泄露。
- **🔄 双线路保障**：支持域名与数字 IP 双重解析，主地址失效时自动切换备用地址。
- **🎨 完美适配**：深度适配原版 UI 及知名联机模组 `mcwifipnp`。

---

## 🛠️ 使用指南

### 1. 开启中转映射 (服务端/房主)
1. 进入单人世界，按 `Esc` 键打开 **对局域网开放**。
2. 将 **“使用中转映射”** 按钮切换为 **“使用”**。
3. 点击 **“创建一个局域网世界”**。
4. 聊天栏将自动弹出外网地址，格式如下：
   > **[MCBridge] 映射成功！**  
   > **域名地址: nat.mrcao.com.cn:XXXXX**  
   > **数字 IP: 103.236.55.246:XXXXX**

### 2. 加入多人游戏 (客户端/好友)
1. 点击主界面的 **多人游戏**。
2. 模组会在后台自动为您开启中转加速链路。
3. 输入房主告知的地址和端口即可加入。

## 📦 安装说明

1. 确保已安装 **Minecraft Forge 1.20.1**。
2. 将本模组的 `.jar` 文件放入游戏的 `mods` 文件夹内。
3. 启动游戏，尽情联机！
