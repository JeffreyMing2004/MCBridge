# 🌐 MCBridge 公共中转节点收集

欢迎来到 MCBridge 的公共节点收集列表！如果您有稳定、开放的 `frps` 服务器资源，并愿意分享给社区，请通过提交 Pull Request 的方式将您的节点信息添加到下方的表格中。

您的贡献将帮助更多玩家轻松地享受联机乐趣！

---

## 提交格式

请严格按照以下 Markdown 表格格式提交，以便我们审核和集成。

| 节点名称 / 地区 | 服务器地址 | 服务器端口 | TLS 支持 | 备注 / 提供者 |
| :--- | :--- | :--- | :--- | :--- |
| 示例：上海电信 | `sh-telecom.example.com` | `7000` | ✅ 是 | 由 [YourName] 提供 |
| | | | | |

**字段说明：**

-   **节点名称 / 地区**：一个具有辨识度的名称，最好包含地理位置和线路信息（如“上海电信”、“美国CN2”）。
-   **服务器地址**：`frps` 服务器的域名或 IP 地址。
-   **服务器端口**：`frps` 的 `bindPort`，用于客户端连接。
-   **TLS 支持**：如果您的服务器强制开启了 TLS (`transport.tls.force = true`)，请填写 ✅ 是；否则填写 ❌ 否。
-   **备注 / 提供者**：您的 GitHub 用户名、网站链接或任何您想展示的简短信息。
